# プロジェクト概要書

## 【案件概要】
ToC向けインタラクティブポートフォリオサイトの開発。
ドラゴンクエストをテーマにしたRPG風のWebアプリケーションで、プロフィール・ポートフォリオ・ブログ・SNS記事集約機能を統合。
サービスフローとして、訪問者がRPGルームでインタラクティブに操作→プロフィール・スキル・実績を閲覧→ポートフォリオプロジェクトを確認→外部記事（Zenn/Qiita/Note）を閲覧→問い合わせにつなげる流れを実現。

## 【担当工程】
新規サービスの立ち上げフェーズでの全工程担当。
技術選定/アーキテクチャ設計/要件定義/実装/テスト/デプロイすべてを担当。

## 【開発手法】
モダンWeb開発（ファイルベースルーティング、コンポーネント駆動開発）

## 【ポジション】
フルスタックエンジニア

## 【コメント】
技術選定、アーキテクチャ設計から実装まで全工程を担当いたしました。
特に、TanStack Routerのファイルベースルーティングを活用することで、型安全で保守性の高いルーティングシステムを実現しました。
また、ドラゴンクエストをテーマにした独自のゲームメカニクス実装により、訪問者に印象的な体験を提供できるポートフォリオサイトを構築しました。
カスタムピクセルフォント、SVGアセット、本格的なタイルベース移動システムなど、細部までこだわった実装を行い、ゲームの懐かしさとモダンWeb開発技術を融合させることができました。
TypeScript strict modeとTailwind CSSを組み合わせることで、型安全性とレスポンシブデザインを両立し、高品質なコードベースを実現しました。
